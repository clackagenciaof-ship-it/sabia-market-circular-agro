# SABIÁ Market

**Sistema Agroalimentar Biointeligente de Água, Alimentos, Adubo e Mercado Local**

> Produzir melhor, vender a tempo, desperdiçar menos e devolver vida ao solo.

## Descrição

O SABIÁ Market é um MVP de plataforma agroalimentar circular que conecta produtores locais, compradores, escola, horta, compostagem e indicadores de impacto.

A solução integra marketplace local, vitrine de excedentes chamada **Última Colheita**, registro de irrigação, reaproveitamento de resíduos orgânicos da merenda escolar e dashboard com métricas sociais, ambientais e econômicas.

## Problema

Pequenos produtores e feirantes enfrentam baixa previsibilidade de demanda, pouca presença digital e risco de perda de alimentos frescos. Ao mesmo tempo, consumidores e escolas têm dificuldade para saber quais alimentos estão disponíveis, onde comprar, quanto custam e de quem estão comprando.

Também existem desperdícios de água na produção e descarte de resíduos orgânicos que poderiam voltar ao solo por meio da compostagem.

## Solução proposta

O MVP demonstra uma plataforma simples, responsiva e navegável com quatro frentes principais:

1. **Mercado Local**: catálogo com produtos, preços, categorias e produtores.
2. **Última Colheita**: vitrine para excedentes, descontos, preço social ou doação orientada.
3. **Água Inteligente**: registro de irrigação e umidade para apoiar o uso racional da água.
4. **BioMerenda**: registro de resíduos vegetais e compostagem escolar.
5. **Dashboard SABIÁ**: indicadores de produtos, pedidos, excedentes, água e resíduos.

## Funcionalidades do MVP

- Catálogo de produtos locais
- Filtro por categoria
- Busca por produto ou produtor
- Cadastro simulado de produto
- Botão de reserva/pedido
- Vitrine Última Colheita
- Registro de irrigação
- Registro de resíduos orgânicos
- Dashboard com indicadores de impacto
- Interface responsiva para celular e computador

## Como executar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/SEU-USUARIO/sabia-market.git
cd sabia-market
```

### 2. Instalar as dependências

```bash
npm install
```

### 3. Rodar o projeto localmente

```bash
npm run dev
```

Depois, acesse o endereço mostrado no terminal, geralmente:

```bash
http://localhost:5173
```

### 4. Gerar versão de produção

```bash
npm run build
```

## Metas de validação

- 5 produtores ou feirantes cadastrados
- 30 produtos cadastrados
- 30 a 50 compradores envolvidos
- 20 pedidos ou reservas gerados
- 20 kg de excedentes aproveitados
- 10% de economia de água no canteiro monitorado
- 15 kg de resíduos vegetais desviados do lixo comum
